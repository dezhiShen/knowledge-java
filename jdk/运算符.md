### 算术运算符
\+ - \* / %
   * %(模,求余)
    任何整数模2不是0就是1，
    所以只要改变被模数就可以实现开关运算。
   * \>> 将整形转为2进制后,将数字向右移动,不分正负数，低位补0;
   ```
      int r=20>>2;
      20的二进制:10100
      20>>2:101
      101转为10进制:5
   ```
   * \<<  将整形转为2进制后,将数字向右移动,不分正负数，低位补0;
   ```
      int r=20<<2;
      20的二进制:10100
      20<<2:1010000
      1010000转为10进制:80

   ```
